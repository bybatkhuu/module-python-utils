To build the python package, run the following command:

```sh
# Install python build dependencies:
pip install -r ./requirements/requirements.build.txt

# Build python package:
python -m build
# Or use the build script:
./scripts/build.sh
```

## Build

```sh
# Install python build:
pip install -U build

# Build help:
python -m build --help
```
